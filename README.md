# Vane Presets Repository

Community-driven DPI bypass rule repository for Vane.

## How to Add a Preset? (Pull Request)

1. Fork this repository.
2. Create a new JSON file in the `community` folder (e.g., `isp-fast-bypass.json`).
3. Enter your rules using the following format:
```json
{
  "id": "unique-id-here",
  "name": "Preset Name",
  "description": "Short description",
  "args": [
    "--arg1",
    "--arg2"
  ]
}
```
4. Submit a Pull Request (PR).

Once approved, it will be automatically added to the main list.

## Nasıl Preset Eklerim? (TR)

1. Bu repoyu fork'layın.
2. `community` klasörü içerisine yeni bir JSON dosyası oluşturun (Örn: `turknet-hizli.json`).
3. Yukarıdaki formatta kurallarınızı girin.
4. Pull Request (PR) gönderin.

Onaylandığında otomatik olarak ana listeye eklenecektir.
